Process

From theory to executable

The verification pipeline ensures correctness at every step.

01

Specify

Define cryptographic primitives as mathematical functions in Isabelle/HOL theories.

definition inner_prod :: "int list => int list => int"
02

Prove

Write machine-checked proofs of correctness. No axioms, no shortcuts.

lemma correctness: "decrypt sk (encrypt pk m) = m"
03

Extract

Generate executable code from verified proofs to your target language.

export_code lwe_encrypt lwe_decrypt in Haskell
