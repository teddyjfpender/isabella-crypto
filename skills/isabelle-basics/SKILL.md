# Isabelle Basics

## Overview

This skill covers the fundamentals of writing Isabelle/HOL theories. Isabelle/HOL is an interactive theorem prover based on Higher-Order Logic, widely used for formal verification of mathematical proofs and software systems. Understanding the basics of theory files, imports, definitions, and lemmas is essential for any Isabelle development.

## Response Checklist

- Theory file structure is correct (header, imports, body, end)
- Import statements include all necessary theories
- Definitions use proper `definition` or `abbreviation` syntax
- Lemmas have clear names and explicit types where needed
- Basic proofs use appropriate methods (auto, simp, etc.)
- Code follows Isabelle naming conventions (snake_case for values, CamelCase for types)
